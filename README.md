Las prácticas contenidas en este repositorio fueron realizadas para la materia de programación avanzada con la Dra. Alinne Michelle. Utilizando y programando el microcontrolador PIC18F45k50 en la tarjeta de desarrollo Miuva.
Con este repositorio no se pretende facilitar el plagio de trabajo entre alumnos, sino ser una referencia de cómo utilizar la biblioteca CPortCOM pues considero es el principal obstaculo para los estudiantes.

Desafortunadamente no pude subir estas carpetas antes, por lo que dos años después se me hace complicado subir una documentación detallada más allá de los comentarios en el código.

Seguramente las carpetas no están bien configuradas y para que funcionen en su computadora en Visual Studio se requieran pasos anteriores, por lo que recomiendo configurar su entorno de desarrollo con los pasos detallados en el PDF proporcionado por la profesora.
